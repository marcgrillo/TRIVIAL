## 1. Abstract
La politica contemporanea è diventata un problema di **User Interface (UI)**: facce, slogan, tonalità di voce, estetica.
Abbiamo smesso di occuparci del **Back-end**: la logica, i processi, le cause e gli effetti.

**TRIVIAL** non è un partito di opinione. È un tentativo di *refactoring* del codice sorgente su cui gira la società.
Rimuoviamo l'interfaccia umana (il leader) per mostrare la struttura nuda dei problemi.

## 2. System Architecture

Il sistema TRIVIAL si basa su un'architettura **Headless**:
* **No Head (Nessun Leader):** Il carisma è un bias cognitivo che altera il giudizio sulla validità di una proposta. Rimuovendo il volto, rimane solo il concetto.
* **No Noise (Nessun Dibattito TV):** Il formato del talk show (finestra temporale limitata + interruzione continua) è incompatibile con la complessità. La verità non si urla in 30 secondi. La verità si spiega.
* **Open Logic:** Le soluzioni non sono "di destra" o "di sinistra". Sono funzionali (il sistema regge) o non funzionali (il sistema crolla).
